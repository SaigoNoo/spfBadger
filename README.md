# spfBadger
spfBadger est une application Python que j'ai crée pour le fun, dont le rôle est d'encoder dans un fichier excel et sur l'écran, tout les passages effectués en insérant la carte d'identité Belge dans un lecteur!

# A propos des données
A savoir, par défaut, la librairie lit la puce, et est donc capable de garder toutes les données, cependant, les seules informations utisées sont:
- Votre numéro de carte d'identité (pas le registre national, ni même le nom prénom)

# Execution
```cmd
python3 main.py
```
